# Emotional Memory Map

Emotional Memory Map is a mobile-first prototype where people pin meaningful stories to a world map. Users tap any location, share a memory with emotional context and an optional photo, then explore a global tapestry of love, loss, growth, and joy.

## Features

- Interactive world map with tap-to-pin behavior
- Story capture form with emotion, title, text, and photo URL
- Memory feed for exploring saved memories
- Pure Expo + React Native implementation for web and mobile

## Run

```bash
cd PROJETS/EmotionalMemoryMap
npm install
npx expo start
```
